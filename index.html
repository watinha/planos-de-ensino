OBJETIVOS

Identificar as tecnologias empregadas em aplicações web. Conhecer as tecnologias e conceitos relacionados ao contexto das aplicações web. Elaborar e organizar aplicações web dinâmicas, interativas e seguras, empregando boas práticas de programação.

ementa

Implantação do serviço HyperText Transfer Protocol (HTTP); Implementação de aplicações HyperText Markup Language (HTML), eXtended Makup Language (XML) e tecnologias associadas. Conceitos de desenvolvimento de aplicações Web para intranet. Desenvolvimento de aplicações Web em camadas; Integração de servidores Web com banco de dados. Linguagens de programação para Web.

procedimentos de ensino
aulas teóricas
As aulas são expositivas utilizando como recursos slides, computador, livros e apostilas. Haverá promoções de discussões sobre os assuntos do programa da disciplina, resolução de exercícios e condução de estudos de caso.
 
aulas práticas
Os temas discutidos nas aulas teóricas sempre terão a sua teoria testada em simulações computacionais de modo que o aluno possa analisar os resultados práticos obtidos. Além disso, serão propostos desafios baseados em cenários reais, nos quais os alunos irão deverão buscar soluções aplicando os conceitos abordados em sala e conhecimento prévio

conteúdo: a recuperação de conteúdo será realizada no decorrer do semestre letivo da seguinte forma:
